# Demo Java Project

A simple Java demo project created to demonstrate basic Java application structure, compilation, and execution.

## Overview 

This repository contains a basic Java application that can be used as a starting point for learning Java, testing Java setup, or demonstrating simple Java code organization.

## Features

* Basic Java project structure
* Simple `main` method example
* Easy to compile and run
* Suitable for beginners and demos

## Project Structure

```text
demo-java/
├── src/
│   └── Main.java
├── README.md
└── .gitignore
```

## Requirements

Before running this project, make sure you have the following installed:

* Java Development Kit 8 or higher
* Git
* Command line or terminal

Check your Java version:

```bash
java -version
```

Check your Java compiler version:

```bash
javac -version
```

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/demo-java.git
```

Navigate into the project folder:

```bash
cd demo-java
```

## Running the Project

Compile the Java file:

```bash
javac src/Main.java
```

Run the application:

```bash
java -cp src Main
```
