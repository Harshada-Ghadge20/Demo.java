## ▶️ How to Run a Java Program

1. Clone this repository:

```bash
git clone https://github.com/Harshada-Ghadge20/Java-Practice.git
```

2. Open the project folder:

```bash
cd Java-Practice
```

3. Compile the Java file:

```bash
javac Main.java
```

4. Run the program:

```bash
java Main
```

> Change `Main.java` and `Main` according to your file and class name.
